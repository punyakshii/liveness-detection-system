# AI Liveness Detection System

## Overview
A Flask-based AI-powered liveness detection system that combines:
- Face Liveness Detection
- Emotion Recognition
- Voice Recognition
- Secure Login Authentication

## Features
✔ Blink Detection
✔ Head Turn Detection
✔ Mouth Open Detection
✔ Emotion Detection
✔ Voice Recognition
✔ Flask Web Dashboard
✔ Real-time Camera Processing

## Tech Stack
- Python
- Flask
- OpenCV
- MediaPipe
- DeepFace
- Vosk
- TensorFlow

pip install -r requirements.txt
